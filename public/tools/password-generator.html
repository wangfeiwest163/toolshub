About Password Security

A strong password should be at least 12 characters long and include a mix of uppercase letters, lowercase letters, numbers, and symbols. Avoid using personal information or common phrases.

Never reuse passwords across multiple accounts. Consider using a password manager to keep track of unique passwords for each account.
