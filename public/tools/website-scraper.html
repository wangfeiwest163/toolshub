How to Use

  1. Enter the URL of the website you want to scrape
  2. Set the maximum number of pages to crawl
  3. Adjust the delay between requests to be respectful to the server
  4. Click "Scrape Website" to start the scraping process
  5. Review the collected data and export if needed

Please be respectful when scraping websites. Always check robots.txt and follow ethical scraping practices.
